# Assets

Esta carpeta contiene los recursos visuales de la aplicación.

## Archivos actuales

Los archivos PNG actuales son placeholders mínimos (1x1 píxel). Para una aplicación en producción, deberías reemplazarlos con:

- **icon.png**: Ícono de la aplicación (1024x1024 px recomendado)
- **splash.png**: Pantalla de carga (1242x2436 px recomendado para iOS)
- **adaptive-icon.png**: Ícono adaptativo para Android (1024x1024 px)
- **favicon.png**: Favicon para web (32x32 o 48x48 px)

## Cómo reemplazar los archivos

1. Crea tus imágenes con las dimensiones recomendadas
2. Reemplaza los archivos en esta carpeta manteniendo los mismos nombres
3. Reinicia el servidor de desarrollo de Expo

## Herramientas recomendadas

- [App Icon Generator](https://www.appicon.co/) - Para generar todos los tamaños de íconos
- [Expo Asset Generator](https://github.com/byCedric/expo-asset-generator) - CLI para generar assets
